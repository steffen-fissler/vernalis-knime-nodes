This plugin contains the original, deprecated PDB Connector KNIME node for advanced 
search and custom reporting of the Protein Data Bank, via RCSB PDB RESTful Web Services.
The node was developed by Enspiral Discovery in collaboration with Vernalis (Cambridge, UK).

The plugin also contains a new family of nodes for querying and generating custom reports 
via the RCSB RESTful Web services, developed by Vernalis (Cambridge, UK)
